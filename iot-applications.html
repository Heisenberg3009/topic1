Developing IoT Applications

Activity 1 of Topic 1 was three major fundamental questions:

1. What is an IOT apllication?
2. What are the main features nedded?
3. What are some examples?

1. What is an IOT application?

It is an application that interconnects devices so that they can communicate with each other eithout using any human interference.

2. What are the main features nedded?

1. Software and Platform
2. Sensors and Devices
3. Communication with the user and other devices connected with.

3. What are some examples?

Some of the examples but not limited to are:
1. Smart Buildings
2. Traffic Monitoring Systems
3. Security Systems
4. Healthcare
5. Inventory Management

Activity 2 of Topic 1 took interest in a speacial case study: Smart Buildings.

It dealt with the questions:
1. What does Smart mean?
2. What are the main features of a Smart Building?
3. What are Smart Building scenarios?

1. What does Smart mean?

Smart has technically many meanings in computer science, but fundamentally it comes to: The ability collect data from lots of sources and grouping them in a single location and then running algorithms to make changes in the real world.

2. What are the main features of a Smart Building?

Some features include but not limited to:
1. Regulation
2. Sensor Tehnology
3. Cloud Computing
4. Centralized Event Processing

3. What are Smart Building scenarios?

1. Temperature Control
2. Automated Lighting
3. Lighting Management
4. Security System
5. Hazard Detection
6. Monitoring Energy Usage
7. Light and Heat Sensors
